# Uno-online

一个基于 **Node.js + Socket.IO** 的联机网页 UNO 游戏。

## 功能

- 大厅：在线人数、房间列表、创建/加入房间
- 房间管理：房主锁房、踢人、移除掉线成员
- 空房间自动销毁
- UNO 游戏核心机制：
  - 反转/跳过/+2/+4
  - 回合方向变化
  - 抢出（可选）
  - +2/+4 叠加（可选）
  - 同色全出（可选）
  - UNO 叫牌与举报（未叫 UNO 罚抽2）
- 结束后排行榜与“再来一局”投票
- 高度自定义设置：
  - +2/+4 是否允许叠加
  - 是否启用同色全出
  - 是否允许抢出
  - 最大人数（2-8）

## 运行

```bash
npm install
npm start
```

打开 `http://localhost:3000`。

## 说明

- 掉线玩家会被标记为离线，30 秒后自动移出房间。
- 若房主离开，房主权限会自动转移给房间内第一位在线玩家。
